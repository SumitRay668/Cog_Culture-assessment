
# Fact-Check Agent

## Features
- Upload PDF
- Extract claims
- Verify claims using Gemini API
- Generate verification report

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deploy easily on Streamlit Cloud.

## Demo Flow
1. Upload PDF
2. Extract claims
3. Click Verify
4. View fact-check results
